# Traffic Sign Recognition System using CNN

## Overview
This project recognizes traffic signs using a Convolutional Neural Network (CNN) built with TensorFlow/Keras. It also includes a Streamlit web application for making predictions.

## Features
- Traffic sign classification
- CNN model trained on the GTSRB dataset
- Streamlit web interface
- Image upload and prediction

## Technologies Used
- Python
- TensorFlow
- Keras
- Streamlit
- NumPy
- Pandas
- OpenCV
- Matplotlib

## Dataset
German Traffic Sign Recognition Benchmark (GTSRB)
